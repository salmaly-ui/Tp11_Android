


# RealTime Tracker

Application Android de geolocalisation en temps reel avec Google Maps.


##  Demo
https://github.com/user-attachments/assets/0eb997a5-3a8a-480c-af5e-b202ee7e15e4

## Fonctionnalites

- Affichage de la carte Google Maps
- Demande automatique de permission de localisation
- Detection de la position GPS et reseau
- Ajout d'un marqueur dynamique sur la position actuelle
- Zoom automatique sur la position
- Boite de dialogue pour activer le GPS si desactive
- Affichage des coordonnees en temps reel

 

### Pre-requis
- Android Studio Hedgehog ou version recente
- SDK Android minimum API 21
- Appareil ou emulateur avec GPS active

### Etapes

1. Cloner ou telecharger le projet
2. Ouvrir le projet dans Android Studio
3. Synchroniser les dependances Gradle
4. Lancer l'application sur un appareil ou emulateur
